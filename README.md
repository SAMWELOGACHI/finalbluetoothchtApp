# Bluetooth Chat Pro
**Secure, Reliable, and Offline Messaging for the Modern Era.**

Bluetooth Chat Pro is a high-performance Android communication platform designed for seamless, internet-free connectivity. Whether you're in a remote location, an emergency situation, or simply want to chat securely without a data plan, Bluetooth Chat Pro provides a premium, feature-rich experience.

![App Logo](app/src/main/res/mipmap-xxxhdpi/ic_launcher_round.png)

## 🌟 Key Features

### 💎 Premium Design
*   **Material 3 Ready**: A sleek, modern user interface following the latest Google design standards.
*   **Midnight Dark Theme**: A sophisticated "Midnight Blue" palette designed for visual comfort and premium aesthetics.
*   **Dynamic UI**: Real-time signal strength indicators and animated scanning states.

### 🗄️ Persistent Chat History
*   **Room Database Integration**: Powered by the Room Persistence Library, all your messages and images are stored locally and securely.
*   **Auto-Restore**: Previous conversations are automatically reloaded whenever you reconnect to a device.

### 📸 Multimedia Sharing
*   **Image Exchange**: Send and receive photos instantly over Bluetooth.
*   **Glide Optimization**: High-performance image rendering and caching for a lag-free experience.

### 🔗 Pro-Link™ Connectivity
*   **Multi-Layered Fallback**: Our intelligent connection algorithm automatically switches between **Secure**, **Insecure**, and **Reflection-based** RFCOMM channels to ensure a 100% successful connection rate across all Android brands.
*   **Foreground Service**: A persistent background service ensures your connection stays alive even when you leave the app.
*   **Intelligent Discovery**: Optimized for Android 13 & 14 (SDK 34), ensuring fast and reliable device scanning.

## 🛡️ Privacy & Security
*   **Zero Data Tracking**: Your messages never leave your device or the local Bluetooth connection. No cloud servers, no data harvesting.
*   **Native Encryption**: Leverages industry-standard Bluetooth encryption protocols for end-to-end local security.

## 🚀 Getting Started
1.  **Enable Bluetooth**: Grant the necessary permissions when prompted.
2.  **Host or Join**: One user clicks **Host Chat**, and the other clicks **Find Devices**.
3.  **Start Chatting**: Select the device from the list and enjoy seamless messaging!
